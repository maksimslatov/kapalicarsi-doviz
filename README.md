# Kapalıçarşı Döviz 🥇

Kapalıçarşı altın ve serbest piyasa döviz kurlarını gösteren, telefona kurulum gerektirmeyen PWA uygulaması.

## Özellikler

- 💵 USD, EUR, GBP, CHF + 6 ek döviz (alış/satış/değişim)
- 🥇 Gram altın, has altın, ons, çeyrek, yarım, tam, Cumhuriyet, gümüş, platin
- ⇄ Kur çevirici (tüm birimler arası)
- 🔄 60 saniyede bir otomatik güncelleme
- 📱 PWA — Chrome'da "Ana ekrana ekle" ile uygulama gibi çalışır

## Veri Kaynağı

[finans.truncgil.com](https://finans.truncgil.com) — ücretsiz, limitsiz, kayıtsız

## Kurulum (GitHub Pages)

1. Bu repoyu fork'la veya klonla
2. **Settings → Pages → Branch: main → Save**
3. `https://kullaniciadi.github.io/kapalicarsi-doviz` adresine git
4. Chrome'da **⋮ → Ana ekrana ekle**

## Lokal Çalıştırma

```bash
git clone https://github.com/KULLANICI/kapalicarsi-doviz
cd kapalicarsi-doviz
npx serve .
# veya
python3 -m http.server 8080
```
